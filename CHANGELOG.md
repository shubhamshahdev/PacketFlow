# Changelog

All notable changes to PacketFlow will be documented in this file.

## [1.0.0] - 2026-06-13

### Added
- Initial release of PacketFlow Network Simulation Platform
- Drag-and-drop network topology builder with 8 device types
- Interactive network connections with VLAN support
- Packet flow visualization with OSI/TCP-IP layer inspection
- Routing simulation (Static, RIP, OSPF)
- Network diagnostics tools (Ping, Traceroute, Port Scanner)
- Real-time network statistics dashboard
- Packet capture viewer with protocol analysis
- Network event logging system
- Topology save/load with import/export (JSON)
- JWT-based authentication system
- Dark and light theme support
- Responsive UI with Tailwind CSS
- RESTful API with Swagger/OpenAPI documentation
- PostgreSQL database with Prisma ORM
- Docker containerization support
- CI/CD pipeline with GitHub Actions
- Comprehensive test suite (Jest + Vitest)
